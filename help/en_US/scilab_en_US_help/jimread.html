jimread

Lit une image à partir d'un fichier ou d'une URL

Syntaxes

image = jimread(imagePath)
[image, imageProperties] = jimread(imagePath)

Arguments

imagePath

Une chaîne de caractère contenant le chemin complet du fichier image ou une URL

image

Une matrice 2D ou 3D

imageProperties

Une liste typée regroupant le titre, la largeur, la hauteur et le type de l'image lue

Description

Cette fonction lit une image à partir de son chemin ou d'une URL. Elle retourne une matrice contenant les données de l'image en niveaux de gris ou une hypermatrice contenant les données RGB de l'image en fonction du type de l'image en entrée.

La liste imageProperties rassemble : le titre de l'image sous forme d'une chaîne de caractères, sa largeur, sa longueur, son type sous forme d'une chaîne de caracères. Le type 'Gray' correspond à une matrice 2D pour les images en niveaux de gris. Le type 'RGB' correspond à une hypermatrice à trois couches. Le type 'RGBA' correspond à une hypermatrice à quatre couches dont la dernière gère la transparence de l'image.

Exemples

//A partir d'un fichier
global jimlabPath
path = jimlabPath + '/help/images/logoEnsim.png';
[im, imProp] = jimread(path);

//A partir d'une URL
path = 'https://webensim.univ-lemans.fr/ressources/icones/interface/Logoensim_2010_tr3.gif';
[image, imProperties] = jimread(path);
